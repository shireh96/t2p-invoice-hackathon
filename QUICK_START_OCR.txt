OPTION A: BASIC SETUP (FREE, 5 MINUTES)
─────────────────────────────────────────────────────────────────────────────

1. Install Tesseract:
   Windows: https://github.com/UB-Mannheim/tesseract/wiki
   → Download tesseract-ocr-w64-setup-v5.3.3.exe
   → Run installer (use default path: C:\Program Files\Tesseract-OCR)
   → Click Next, Next, Install

2. Install Python packages:
   pip install pytesseract Pillow pdf2image

3. Configure:
   copy .env.example .env
   → Edit .env file, set: OCR_BACKEND=tesseract

4. Restart server:
   start_server.bat

DONE! Upload an invoice to test.

TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────

Problem: "Tesseract not found"
Solution: Add to .env file:
   TESSERACT_CMD=C:\\Program Files\\Tesseract-OCR\\tesseract.exe

Problem: "Cannot convert PDF"
Solution: Install Poppler for Windows:
   1. Download: https://github.com/oschwartz10612/poppler-windows/releases/
   2. Extract to C:\poppler
   3. Add C:\poppler\Library\bin to Windows PATH

Problem: Server shows "Using simulated OCR"
Solution: Dependencies not installed. Run:
   pip install pytesseract Pillow pdf2image

Problem: AI not working
Solution: Check your API key in .env file and verify:
   pip list | findstr anthropic
