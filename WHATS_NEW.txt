✅ INSTALLATION STEPS
─────────────────────────────────────────────────────────────────────────────

STEP 1: Install Python packages
   → Run: install_with_openai.bat
   → Or:  pip install flask flask-cors pytesseract Pillow pdf2image openai

STEP 2: Install Tesseract OCR
   → Download: https://github.com/UB-Mannheim/tesseract/wiki
   → Run installer (use default path)
   → Verify: tesseract --version

STEP 3: (Optional) Install Poppler for PDF support
   → Download: https://github.com/oschwartz10612/poppler-windows/releases/
   → Extract to C:\poppler
   → Add C:\poppler\Library\bin to PATH

STEP 4: Start the server
   → Run: start_server.bat
   → Look for: "✓ Enhanced OCR initialized: backend=tesseract, AI=openai"

STEP 5: Test it!
   → Open: http://localhost:5000
   → Upload an invoice
   → Watch AI extract data automatically!
